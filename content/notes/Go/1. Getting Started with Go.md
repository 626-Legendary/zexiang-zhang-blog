# Getting Started with Go

> Go 语言入门笔记（整理版 · 完整版）

---

## 1. 为什么学习 Go（Why Go）

- **编译型语言**，性能接近 C/C++
- 内置 **垃圾回收（GC）**，开发效率高
- 原生支持 **并发（Concurrency）**
- 语法简洁，非常适合：
  - 后端服务
  - 分布式系统
  - 云原生 / DevOps / CLI 工具

---

## 2. Go 程序结构

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go")
}
```

- `package main`：可执行程序入口包
- `func main()`：程序入口函数

---

## 3. Go Toolchain（常用命令）

| 命令 | 作用 |
|---|---|
| go run | 编译并运行 |
| go build | 编译 |
| go fmt | 自动格式化 |
| go test | 运行测试 |
| go mod init | 初始化模块 |

---

## 4. 变量与常量

### 4.1 变量声明

```go
var x int = 10
x := 20 // 仅函数内
```

### 4.2 零值（Zero Value）

| 类型 | 零值 |
|---|---|
| int | 0 |
| string | "" |
| bool | false |
| pointer/slice/map | nil |

### 4.3 常量与 iota

```go
const (
    A = iota
    B
    C
)
```

---

## 5. 基本数据类型

- 整数：int, int64, uint
- 浮点：float32, float64
- 布尔：bool
- 字符：rune（Unicode）
- 字符串：string（不可变）

---

## 6. 控制结构

### if
```go
if x > 0 {
}
```

### for（唯一循环）
```go
for i := 0; i < 10; i++ {}
for x < 10 {}
for {}
```

### switch
```go
switch x {
case 1:
case 2:
default:
}
```

---

## 7. 指针（Pointers）

```go
x := 10
p := &x
*p = 20
```

- Go 有指针，但没有指针运算
- 常用于：
  - 修改外部变量
  - 避免拷贝大对象

---

## 8. 复合类型

### 数组（Array）
- 固定长度
- 很少直接使用

### 切片（Slice）

```go
s := make([]int, 0, 10)
s = append(s, 1, 2)
```

- len：当前长度
- cap：底层数组容量

### 映射（Map）

```go
m := make(map[string]int)
m["a"] = 1
```

---

## 9. Struct（结构体）

```go
type Person struct {
    Name string
    Age  int
}
```

---

## 10. 内存模型：Stack vs Heap

- 栈：函数调用生命周期
- 堆：逃逸对象，由 GC 管理

```go
func foo() *int {
    x := 1
    return &x // 逃逸到堆
}
```

---

## 本章总结

- Go 是高性能 + 高效率语言
- 熟悉变量、控制流、切片、map 是关键
- 理解零值、指针、内存模型非常重要
